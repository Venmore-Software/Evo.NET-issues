# Evo.NET-issues
Public issue tracker for [Evo.NET, Web Interface for Sage 200 Evolution](https://evo.net.za/).

## Guidelines for Posting Issues

* Please include the version number. This is visible in the login screen and in the status bar at the bottom right of the Evo.NET application.
* Please create a separate issue for each item (don't use one issue to report multiple problems).
* Avoid attaching documents whose content could be included in the issue directly (e.g. a .docx that contains descriptions and screenshots). Add this content directly to the issue.
